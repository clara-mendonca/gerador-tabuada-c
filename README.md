# Gerador de Tabuada em C

Projeto simples desenvolvido para praticar lógica de programação utilizando a linguagem C.

## Funcionalidade
O usuário informa um número e o programa gera a tabuada de 1 a 10.

## Tecnologias
- Linguagem C

## Objetivo
Projeto de prática desenvolvido durante os estudos de programação no curso de Análise e Desenvolvimento de Sistemas.
